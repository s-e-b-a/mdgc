**Inventario**

1. Requerimientos de Datos por Categoría
A. Para Videojuegos (Físicos y Digitales)

Título: Nombre completo del juego.

Plataforma: Consola o sistema (PS5, PC, Switch, etc.).

Formato: Selección entre "Físico" o "Digital".

Estado de completitud: (Nuevo/Sellado, Completo en caja, Solo cartucho/disco).

Región: (NTSC, PAL, Asia).

Tienda/Origen: Donde se compró (Steam, eShop, tienda física específica).

Precio de compra: Valor pagado.

Fecha de adquisición: Cuándo se sumó a la colección.

Estado de juego: (Pendiente, Jugando, Completado, Abandonado).

B. Para Consolas (Hardware)

Modelo: (Ej: PS4 Pro, Nintendo Switch OLED).

Número de Serie: Para control de garantía o robo.

Color/Edición: (Ej: Edición especial Zelda, Color Negro).

Estado: (Funcional, Requiere reparación, Para repuestos).

Almacenamiento: Capacidad del disco duro o memoria interna.

Cables incluidos: (HDMI, de poder, AV).

C. Para Accesorios

Tipo: (Control, Volante, Headset, Memory Card).

Marca: (Original/First-party o Genérico/Third-party).

Conectividad: (Alámbrico, Bluetooth, Radiofrecuencia).

2. Requerimientos Funcionales del Sistema
Para que el sistema sea útil en el día a día, debería cumplir con lo siguiente:

Búsqueda y Filtrado: Poder buscar por título o filtrar por consola (ej: "ver todos mis juegos de retrocompatibilidad").

Lectura de Código de Barras: (Opcional pero recomendado) Integración con cámara para escanear el código de barras de juegos físicos y auto-completar la información.

Gestión de Préstamos: Un apartado para anotar a quién le prestaste un juego y en qué fecha.

Galería de Imágenes: Posibilidad de subir fotos reales de tus artículos (útil para seguros o para mostrar el estado de conservación).

Estadísticas y Valoración:

Cálculo automático del valor total de la colección.

Gráficos por plataforma o formato.

Exportación de Datos: Capacidad de exportar a Excel, CSV o PDF para tener respaldos externos.
